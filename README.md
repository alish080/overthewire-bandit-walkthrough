# OverTheWire Bandit Walkthrough

This repository contains my personal walkthrough and notes while solving the Bandit wargame from OverTheWire.

I am using this challenge to strengthen my Linux fundamentals, command-line skills, and problem-solving ability.

---

## What is Bandit?

Bandit is a beginner-friendly wargame designed to teach the basics of:

- Linux commands
- File system navigation
- SSH usage
- Permissions
- Text processing
- Basic security concepts

Official website:
https://overthewire.org/wargames/bandit/
